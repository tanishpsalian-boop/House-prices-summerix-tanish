This project focuses on predicting house prices using the California Housing Dataset and a Linear Regression model. The dataset was preprocessed by handling missing values and encoding categorical data. Exploratory Data Analysis (EDA) was performed to understand feature relationships and data distribution. The model was trained using 80% of the data and evaluated on the remaining 20% using MAE, RMSE, and R² Score. The results show that the model is able to predict house prices with reasonable accuracy, with median income being the most influential feature.
